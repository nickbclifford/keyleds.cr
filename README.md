# keyleds.cr

A Crystal interface to the [`libkeyleds`](https://github.com/keyleds/keyleds) library.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     keyleds:
       github: nickbclifford/keyleds.cr
   ```

2. Run `shards install`
